### Retrieval-Augmented Generation (RAG)

**RAG (Retrieval-Augmented Generation)** ‚Äî —ç—Ç–æ —Ç–µ—Ö–Ω–æ–ª–æ–≥–∏—è, –∫–æ—Ç–æ—Ä–∞—è –¥–∞–µ—Ç –Ω–µ–π—Ä–æ—Å–µ—Ç–∏ ¬´–≤–Ω–µ—à–Ω—é—é –ø–∞–º—è—Ç—å¬ª –∏–ª–∏ ¬´—Å–ø—Ä–∞–≤–æ—á–Ω–∏–∫¬ª, —á—Ç–æ–±—ã –æ–Ω–∞ –ø–µ—Ä–µ—Å—Ç–∞–ª–∞ –≥–∞–ª–ª—é—Ü–∏–Ω–∏—Ä–æ–≤–∞—Ç—å –∏ –º–æ–≥–ª–∞ –æ—Ç–≤–µ—á–∞—Ç—å –Ω–∞ –æ—Å–Ω–æ–≤–µ —Å–≤–µ–∂–∏—Ö –∏–ª–∏ –∑–∞–∫—Ä—ã—Ç—ã—Ö –¥–∞–Ω–Ω—ã—Ö.

![RAG_diagram](https://github.com/Rakhmankulov/Interviews/blob/main/images/RAG_diagram.jpg)

–ï—Å–ª–∏ –ø—Ä–æ—Å—Ç–∞—è LLM ‚Äî —ç—Ç–æ **—Å—Ç—É–¥–µ–Ω—Ç –Ω–∞ —ç–∫–∑–∞–º–µ–Ω–µ**, –∫–æ—Ç–æ—Ä—ã–π –ø–æ–ª–∞–≥–∞–µ—Ç—Å—è —Ç–æ–ª—å–∫–æ –Ω–∞ —Å–≤–æ—é –ø–∞–º—è—Ç—å, —Ç–æ RAG ‚Äî —ç—Ç–æ **—Ç–æ—Ç –∂–µ —Å—Ç—É–¥–µ–Ω—Ç, –∫–æ—Ç–æ—Ä–æ–º—É —Ä–∞–∑—Ä–µ—à–∏–ª–∏ –ø–æ–ª—å–∑–æ–≤–∞—Ç—å—Å—è –±–∏–±–ª–∏–æ—Ç–µ–∫–æ–π**.

### –ö–∞–∫ —ç—Ç–æ —Ä–∞–±–æ—Ç–∞–µ—Ç (3 –ø—Ä–æ—Å—Ç—ã—Ö —à–∞–≥–∞):

2.  **–î–æ–ø–æ–ª–Ω–µ–Ω–∏–µ (Augmented):** –ù–∞–π–¥–µ–Ω–Ω—ã–µ –∫—É—Å–∫–∏ —Ç–µ–∫—Å—Ç–∞ (–∫–æ–Ω—Ç–µ–∫—Å—Ç) –ø—Ä–∏–∫–ª–µ–∏–≤–∞—é—Ç—Å—è –∫ –≤–∞—à–µ–º—É –≤–æ–ø—Ä–æ—Å—É. –ü–æ–ª—É—á–∞–µ—Ç—Å—è –¥–ª–∏–Ω–Ω—ã–π –ø—Ä–æ–º–ø—Ç: *¬´–ò—Å–ø–æ–ª—å–∑—É—è –≤–æ—Ç —ç—Ç—É –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏—é: [–Ω–∞–π–¥–µ–Ω–Ω—ã–π —Ç–µ–∫—Å—Ç], –æ—Ç–≤–µ—Ç—å –Ω–∞ –≤–æ–ø—Ä–æ—Å: [–≤–∞—à –≤–æ–ø—Ä–æ—Å]¬ª*.
3.  **–ì–µ–Ω–µ—Ä–∞—Ü–∏—è (Generation):** –ù–µ–π—Ä–æ—Å–µ—Ç—å —á–∏—Ç–∞–µ—Ç –ø—Ä–µ–¥–æ—Å—Ç–∞–≤–ª–µ–Ω–Ω—ã–π —Ç–µ–∫—Å—Ç –∏ –≤—ã–¥–∞–µ—Ç —Ç–æ—á–Ω—ã–π –æ—Ç–≤–µ—Ç, –æ–ø–∏—Ä–∞—è—Å—å –Ω–∞ —Ñ–∞–∫—Ç—ã –∏–∑ ¬´—Å–ø—Ä–∞–≤–æ—á–Ω–∏–∫–∞¬ª.

### –ü–æ—á–µ–º—É —ç—Ç–æ –∫—Ä—É—Ç–æ:
*   **–ê–∫—Ç—É–∞–ª—å–Ω–æ—Å—Ç—å:** –ù–µ –Ω—É–∂–Ω–æ –ø–µ—Ä–µ–æ–±—É—á–∞—Ç—å –º–æ–¥–µ–ª—å, —á—Ç–æ–±—ã –æ–Ω–∞ —É–∑–Ω–∞–ª–∞ –æ —Å–µ–≥–æ–¥–Ω—è—à–Ω–∏—Ö –Ω–æ–≤–æ—Å—Ç—è—Ö ‚Äî –¥–æ—Å—Ç–∞—Ç–æ—á–Ω–æ –æ–±–Ω–æ–≤–∏—Ç—å –±–∞–∑—É –¥–æ–∫—É–º–µ–Ω—Ç–æ–≤.
